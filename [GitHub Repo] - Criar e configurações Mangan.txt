No PC, criar pasta para clonar projeto criado no GitHub.

Canto direito central inferior: link para clone na pasta desejada.

[Se projeto novo] Criar projeto Eclipse diretamente dentro da pasta de clone (master).

Criar classes e etc, e subir com git push pro repositorio git hub.

Adicionar em 'issues' tarefas a serem trabalhadas. [git commit -m "#1 mensagem etc"]

Pra compartilhar projeto, usar clone do canto e subir após. Mas para funcionar, é preciso add (==autorizar) contribuidores.

DICAS:
Fork: Canto superior direito. Possibilita adquirir uma cópia do projeto e seguir trabalhando a partir das ultimas alterações. Porem ao subir, é necessario fazer PullRequest (pelos dois) pra poder juntar ao master onde todos tem acesso.
